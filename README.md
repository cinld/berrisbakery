# Berri's Bakery

**Berri’s Bakery** is a single player 2D game developed in **Unity** using **C#**, featuring item collection and power-up mechanics within a baking-themed gameplay loop.

## Overview
The player controls Berri, navigating through levels to collect ingredients required for baking while utilizing power-ups to enhance performance and efficiency. The game emphasizes responsive movement, object interaction, and lightweight resource management.

## Features
- **Ingredient Collection System**  
  Uses trigger-based collision detection to collect items and update inventory state in real time.

- **Power-Up Mechanics**  
  Implements temporary gameplay modifiers (e.g., speed boosts, score multipliers) via modular scripts and coroutine-based timers.

- **Single Player Gameplay Loop**  
  Structured around continuous progression with increasing difficulty and dynamic item spawning.

- **Physics & Input Handling**  
  Leverages Unity’s built-in physics engine for collision detection and a frame-based input system for player control.

## Technical Details
- **Engine:** Unity  
- **Language:** C#  
- **Architecture:** Component-based design using `MonoBehaviour` scripts  

### Core Systems
- **Player Controller:** Handles movement, input, and interaction logic  
- **Item System:** Manages spawning, collision detection, and collection  
- **Power-Up System:** Controls state changes and duration effects  
- **UI System:** Updates score, inventory, and player feedback in real time
